## Purpose

Use this section to understand the main SPRK layout, move between pages, and find records quickly without guessing where they live.

## Prerequisites

- You can sign in to SPRK.
- An active company is already selected.

## Steps

1. Open the app and confirm you can see the sidebar, the global search bar, and the main content area.
2. Start with the workflow that matches what you need:
   - [Understand the dashboard overview](./understand-the-dashboard-overview.md)
   - [Move between major app areas](./move-between-major-app-areas.md)
   - [Use global search](./use-global-search.md)
   - [Understand company-aware navigation](./understand-company-aware-navigation.md)

## Expected Result

You can recognize the major navigation controls in SPRK and use them to reach the right page quickly.

## Common Mistakes

- Treating the dashboard as the only navigation entry point. The sidebar and search bar are both primary navigation tools.
- Forgetting that the active company affects what data appears throughout the app.
- Assuming every search result opens the same page. Different result types route to different destinations.
