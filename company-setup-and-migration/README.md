## Purpose

Use this section when you need to create a company, move setup data into SPRK, or change which company you are working in.

## Prerequisites

- You can sign in to SPRK.
- You can open `Settings` and the `Companies` tab.
- If you plan to import, your source files are already exported from the previous system.
- If your workspace already has a real company, your license allows another company to be created.

## Steps

1. Open `Settings`.
2. Select the `Companies` tab if it is not already active.
3. Choose the workflow that matches your goal:
   - [Create your first company](./create-your-first-company.md)
   - [Import from QuickBooks Online ZIP](./import-from-quickbooks-online-zip.md)
   - [Import from QuickBooks Desktop IIF](./import-from-quickbooks-desktop-iif.md)
   - [Use the Import Wizard](./use-the-import-wizard.md)
   - [Switch between companies](./switch-between-companies.md)

## Expected Result

You can reach the right company setup or migration path without guessing which action to use.

## Common Mistakes

- Starting from another page and missing the `Companies` tab in `Settings`.
- Creating a new blank company when you meant to import historical setup data.
- Treating `Import from QBO (ZIP)` and `Import from QBD (IIF)` as interchangeable. They expect different file formats.
