# Understand the Dashboard at a High Level

Use the dashboard for a quick 30-day snapshot of business activity and as a starting point for deeper workflows.

![SPRK Dashboard in Demo Company showing 30-day summary cards, cash movement, expenses by category, and Quick Links](../screenshots/getting-started/dashboard-at-high-level-step-01.png)

## Purpose

Use this workflow when you first enter SPRK and want a quick read on recent company activity before drilling into detailed pages.

## Prerequisites

- An active company is selected.
- The sidebar is visible.

## Steps

1. Open `Dashboard` from the `General` section of the sidebar.
2. Review the top summary cards:
   - `Income (30d)`
   - `Expenses (30d)`
   - `Profit (30d)`
   - `Ledger Lines (30d)`
3. Review the chart panels:
   - `Cash Movement (30d)`
   - `Expenses by Category (30d)`
4. Use `Quick Links` if you want to jump directly into a common page such as `Reports`, `Settings`, or `Invoices`.
5. Move into the detailed page that matches the question you want to answer next.

## Expected Result

You can use the dashboard as a summary and launch point instead of as a transaction-entry page. Current general ledger impact as of 2026-05-04:

- Opening the dashboard does not create a journal entry.
- Dashboard cards and charts read existing posted data for the active company.
- Quick links are navigation controls only.

## Common Mistakes

- Treating dashboard values as the place to investigate record-level detail.
- Using dashboard numbers without first confirming the active company.
- Expecting the dashboard to replace reports, ledger review, or page-specific workflows.
