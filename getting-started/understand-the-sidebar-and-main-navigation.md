# Understand the Sidebar and Main Navigation

Learn how the sidebar sections, app header, global search, and main content area work together in SPRK.

## Purpose

Use this article when you want to understand the main SPRK layout before you start working in detailed accounting pages.

## Prerequisites

- You are signed in to SPRK.

## Steps

1. Review the left sidebar as the main page navigation for the app.
2. Use the company selector at the top of the sidebar to confirm the active company.
3. Use the `General`, `Accounting`, and `System` sidebar groups to move between the major app areas.
4. Use the search bar in the app header when you want to search customers, vendors, or ledger records directly.
5. Treat the main content panel as the working area for the page you selected from the sidebar.
6. Use the sidebar collapse control if you want a narrower navigation view.

## Expected Result

You can recognize where to change pages, where to confirm company context, and where to search without guessing. Current general ledger impact as of 2026-05-04:

- Navigation actions do not post to the general ledger.
- Search and page switching only change what you are viewing.

## Common Mistakes

- Assuming the dashboard is the only place to start a workflow.
- Forgetting that global search results route to different pages depending on the result type.
- Treating navigation changes as saved accounting actions.
